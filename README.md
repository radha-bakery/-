# Radha Bakers Website

Static GitHub Pages website for Radha Bakers.

## Upload to GitHub
Upload these files/folders at repository root:

- index.html
- style.css
- script.js
- products.json
- README.md
- assets/

Do not upload the ZIP directly. Extract it first.

## Features
- Premium UI/UX
- AI realistic product photos
- Product categories
- Add to Cart
- Quantity control
- Variant pricing for Full/Half items
- WhatsApp checkout to 9418310089
- Mobile responsive
- GitHub Pages ready

## GitHub Pages
Settings → Pages → Branch: main → Folder: /root → Save
